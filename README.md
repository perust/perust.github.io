### 3. 숫자야구

종이와 연필을 가지고 할 수 있는 간단한 게임.
상대방이 생각한 3개 또는 4개의 숫자가 무엇인지 맞추는 게임이다.
숫자 자릿수까지 맞춰야하며, 숫자만 맞추면 n(갯수)볼, 숫자와 자리까지 맞추면 n스트라이크로 힌트를 준다. 번갈아가며 진행한다.
